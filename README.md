# Spectra Music Player
Native Android Music Player which streams from Hi-Fi

  ## Features

  - **Streaming** - Integrated Hi-Fi support for high-quality streaming
  - **Lossless Tracks** - All songs are Lossless FLAC tracks
  - **No API keys needed** - Doesn't ask for any permissions or api keys
  - **Smart Library** - Browse by tracks, albums, artists
  - **Playlist Management** - Create custom playlists, manage favorites, and track your history
  - **Spotify Import** - Import your Spotify playlists with automatic track matching to TIDAL
  - **Smart Queue** - Play next, add to queue, and intelligent queue management
  - **Beautiful UI** - Material Design 3 with smooth animations and dark/light themes
  - **Background Play** - Control playback from notification and lock screen

  ## Key Features Explained

  ### Spotify Playlist Import
  Import your Spotify playlists and automatically convert them into High Quality FLACS. Imports happen in the background so you can keep using the app.

  ### Streaming Integration
  Direct support for HiFi API with automatic URL resolution and caching for fast playback.
